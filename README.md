# satellite-image-classification-dl
# Satellite Image Enhancement and Land Classification

## Description
This project focuses on processing satellite images and classifying land types such as forest, water, and urban areas using machine learning techniques.

## Features
- Image enhancement
- Feature extraction
- Land classification

## Tools Used
- Python
- OpenCV
- NumPy

## How to Run
1. Install dependencies
2. Run the main file
3. Provide input image
4. Get output classification

## Conclusion
This project demonstrates how machine learning can be applied in real-world satellite image analysis.
